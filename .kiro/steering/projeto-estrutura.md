# Estrutura do Projeto

## Arquivos Principais

- **`index.html`** - Aplicação principal (jogo de bomba 3D)
- **`styles.css`** - Estilos e efeitos 3D
- **`script.js`** - Lógica completa do jogo

## Arquivos de Teste (Ignorar)

- `cube-simple.html` - Teste simples
- `instrucoes.html` - Teste de instruções

## Resumo

Projeto de jogo web com cubo 3D interativo, mini-jogos de desarmamento de bomba, cronômetro e efeitos visuais avançados. Foco nos 3 arquivos principais apenas.
